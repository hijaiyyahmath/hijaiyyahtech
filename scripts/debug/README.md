# Tracing & Manual Debugging
Location: `scripts/debug/`

Utilities for manual inspection, debugging, and trace generation during development.

## Key Scripts
- `debug_jeem.py` / `extract_jeem.py`: Special case handling for 'Jim' character logic.
- `hc18dc_trace_v01.py` / `hc18dc_jeem_trace_v01.py`: Trace generators for audit evidence.
- `hc18dc_enc_tv_runner.py`: Logic for testing encryption/trace vectors.
- `hc18dc_tamper_test.py`: Deliberate tampering tests to verify verifier robustness.
