# Artifact & Table Generators
Location: `scripts/generators/`

Tools in this folder are responsible for synthesizing data artifacts from raw sources.

## Key Scripts
- `gen_table.py` / `create_audit_table.py`: Generates the formatted ISA and Feature tables.
- `generate_MH_18x28.py`: Logic for creating the 18x28 feature matrix.
- `generate_st28.py` / `generate_st28_full.py`: Compiles the ST-28 normative dataset.
- `generate_hl_artifacts_v2.py`: Pipeline for creating HijaiyahLang-specific artifacts.
